# Eturem

Easy To Understand Ruby Error Message.

[日本語説明（こちらの方が詳細）](https://github.com/nodai2hITC/eturem/blob/master/README.ja.md)

## Installation

install it yourself as:

    $ gem install eturem

## Usage

    $ ruby -returem <your_script.rb>
